This is a production-ready Next.js + TypeScript portfolio scaffold with optional Supabase Admin.

## Quick start

1. Install:
```bash
npm install
```

2. Create `.env.local` from `.env.example` and fill required keys.

3. Run locally:
```bash
npm run dev
```

4. To sync Supabase -> local JSON (manual):
```bash
npm run sync:supabase
```

5. Deploy: Connect to Vercel and add env vars.
